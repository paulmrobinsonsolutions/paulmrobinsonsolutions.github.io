EPMD is an Excel-based "live" monitoring tool to visually see the current health of the process automation system. It provides a schedule-by-schedule view of the current status of all processes, allowing for quick identification and remediation of any issues that may arise. This Excel file uses a direct read-only, non-locking query to get the necessary data for multiple worksheets. Formulas and conditional formatting is then used0 via VBA code to apply
Key Technologies: Power Query, Excel VBA, and Excel formulas (Vlookup, Find, If, etc.), and conditional formatting.

The Power Apps File Upload was created as a way to avoid direct SharePoint access with the user managements aspect. This was a collaboration with a colleague to create a Power App that allows users to upload files directly to a SharePoint library with several validation checks in place like file size, file type, limited number of files, and more. The cool thing is that the list of options to select from are dynamically loaded from other SharePoint lists, decoupling the file upload app from the list options so the app does not need to be modified going forward. As list options are added and/or removed, the app will automatically reflect those changes.
Key Technologies: Power Apps, Power Automate, SharePoint, and DAX
